MVC Cart demo

This is simple open source cart system based on MVC pattern with MySQL database. It contains different functions for users, but no functions for administrator. Most important functions are

  • Register
  • Login
  • Verify email for new users
  • Display items
  • Add items to cart
  • Remove items from cart
  • Change quantity of an item
  • Place order
  • Search an item
  • Sort items
  • Paging

You can download the cart system on github under https://github.com/spikir/cart_mvc
